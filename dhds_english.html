General

In the show(s) we see 5 varieties of dialing devices:

  1. Milkyway
  2. Pegasus
  3. Atlantis
  4. Puddle jumper
  5. Universe Remote

DHDs are the primary method of dialing stargates in the show they connect to the nearest gate through a subspace connection. Within planetary DHDs there is a Cold fusion reactor designed to last for thousands of years to power the connected gate

Universe Remote

The universe remote is not actually a DHD, in the show it communicated with the ramp of the nearest planetary Universe gate. In the mod instead of communicating with the ramp the Universe Remote directly communicates with the gate. The Universe remote has 4 modes:

  1. Nearby
  2. Memory
  3. Rings
  4. OC

The nearby mode displays and allows you to dial the universe addresses of all gates within 1024 blocks(configurable). The Memory mode allows you to dial any address you have saved to your dialer in the same way as you record an address on a notebook page. The rings mode allows you to control the nearest set of transport rings to your location. The OC mode allows you to broadcast a wireless OC (open computers) message to a specific port to be picked up either by a gate to be relayed through an active wormhole or be picked up by an OC (open computers) computer you can also specify a modem address so that it is only sent to a specific computer.

Milkyway

The Milkyway DHD by default is only able to dial 7 Glyph address however can be upgraded to be able to dial up to 9 glyph addresses using a DHD glyph crystal. the DHD has 39 buttons on it, 38 of them for Glyphs and the 39th button is the BRB (Big Red Button) used to engage a wormhole.

Pegasus

The Pegasus DHD by default is only able to dial 7 Glyph address however can be upgraded to be able to dial up to 9 glyph addresses using a DHD glyph crystal. the DHD has 39 buttons on it, 38 of them for Glyphs (2 glyphs that go unused) and the 39th button is the BBB/BGB (Big Blue Button/Big Green Button) used to engage a wormhole.
